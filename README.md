# SameLobby

SameLobby is an adult gamer friendship and teammate platform built around compatible availability, clear expectations, and repeat play.

## Testing

See [TESTING.md](./TESTING.md) for suite commands, local Supabase setup, and traceability. Active test-program work lives on branch `test-program/planning-v4` (baseline verification at `docs/testing/baseline/0522b2d/`).

## Technology

- Next.js
- TypeScript
- Supabase Auth and PostgreSQL
- Vercel

Development begins with **Slice 1.1: true login and account creation**.
