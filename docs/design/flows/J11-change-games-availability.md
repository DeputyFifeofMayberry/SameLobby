# J11 — User changes games or availability

**Journey:** 19.11 | **Screens:** Profile → Games → Availability → Intent → Teammates

## User goal

Keep discovery and relationships accurate when life or interests change.

## Flow

```
Profile editor tabs
  → Activate/archive games; update platforms
  → Edit recurring windows OR mark temporarily unavailable
  → Pause discovery (one control)
  → Update or renew current intent (expires 14 days)
  → Existing connections/groups preserved
```

## System response

- Refresh recommendations
- Show teammates new shared games without broadcasting private schedule
- Monthly availability review prompt (optional)

## Success

Discovery stays current; relationships continue across change.
