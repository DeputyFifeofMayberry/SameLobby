# J04 — Parent finds interruption-friendly gamers

**Journey:** 19.4 | **Screens:** Profile preferences → Discover → Profile → Connect → Play

## User goal

Find people tolerant of interruptions and shorter sessions **without** disclosing family details.

## Flow

```
Set environment: interruption-friendly + typical session length + schedule window
  → Optionally: interruption-tolerant preference (match-only)
  → Discover / filter
  → Connect and discuss expectations
  → Play invitation with realistic session length
```

## Copy rules

- Tag describes **patience with pauses**, not identity verification
- Never require children, ages, names, or household info
- Doxxing and personal-info requests prohibited (reportable)

## Visibility

Preference shown only per user's visibility setting; never infer parenthood.

## Success

Both users play with aligned expectations about pauses and availability.
