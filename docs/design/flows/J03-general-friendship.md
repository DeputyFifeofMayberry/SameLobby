# J03 — User looks for general gaming friendship

**Journey:** 19.3 | **Screens:** Discover → Friendship intent → Recommendations → Profile → Request → Messages

## User goal

Meet adults for conversation and future multi-game play, not only filling one lobby.

## Flow

```
Select "gaming friendship" as current goal
  → Optional: preferred games/genres, social energy, communication, schedule, group size, interests
  → Browse limited recommendation set (≤12)
  → Connect via request
  → After mutual acceptance: message with icebreakers
```

## System response

- Broader game/genre overlap + interests
- Retain platform, schedule, language, safety, hard-environment rules
- Each result shows ≥3 concrete reasons + suggested icebreaker

## Notes

- Short-lived current intent optional
- General availability match-only and optional
- No public friend counts or popularity signals

## Success

Comfortable conversation with path to play invitation in any shared game later.
