# J05 — Clean-communication user finds compatible teammates

**Journey:** 19.5 | **Screens:** Environment preferences → Discover → Connect → Play

## User goal

Find people willing to maintain requested communication boundaries.

## Flow

```
Select environment preferences (e.g. minimal profanity as hard OR strong preference)
  → Optional: no hostility toward religion/nonreligion, alcohol/drug-free conversation
  → Review relevant recommendations with alignment facts
  → Connect; acknowledge hard boundaries if applicable
  → Play
```

## Rules

- Neutral environment language
- No religion, recovery, health, or family disclosure required
- Hard boundaries may require acknowledgement before play invite
- Never display why user selected preference

## Success

Conversation and play with visible, acknowledged boundaries.
