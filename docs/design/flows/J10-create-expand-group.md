# J10 — User creates or expands a small teammate group

**Journey:** 19.10 | **Screens:** Teammates → Create group → Members → Group chat

## User goal

Turn compatible connections into a private duo, trio, squad, or friend group.

## Flow

```
Teammates → Create group
  → Name + abstract emblem
  → Add 2–7 existing connections (invite)
  → Set shared games, communication expectations, size goal
  → Each invitee accepts individually
  → Groups ≤4: unanimous approval for new members; larger: majority
  → Group chat + play invitations enabled
```

## Constraints

- 3–8 members max
- Invitation-only; no public directory
- One complete private group free per user
- Owner + up to 2 admins; any member may leave immediately

## Safety

- Cannot add blocked users
- Report group or member
- Admin actions logged

## Success

Stable home for chat, shared games, availability, play plans.
