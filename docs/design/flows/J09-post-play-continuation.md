# J09 — Users continue the relationship after playing

**Journey:** 19.9 | **Screens:** Play completion → Continuation → Teammate detail

## User goal

Make continued contact easy without exposing rejection or ratings.

## Flow

```
After session window: private prompt per user
  → "Did play occur?" (optional)
  → Choices: keep chatting, play again, add teammate, add to group, not now, report issue (separate path)
  → Mutual teammate choice updates relationship only when both agree
  → Positive mutual progression only revealed
```

## Copy

"Keep this connection going?" — neutral, not evaluative.

## Frequency

Once after first confirmed play; later only on user request or group change.

## Prohibited

Stars, public reviews, rejection reasons, desirability scores.

## Success

Easier re-invite; shared play history private to participants.
