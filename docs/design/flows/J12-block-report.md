# J12 — User blocks or reports another member

**Journey:** 19.12 | **Screens:** Any surface menu → Block/Report → Safety Center

## User goal

Stop contact immediately; credible safety response for reports.

## Flow

```
From profile, request, chat, invitation, teammate, or group:
  → Block (confirm) — immediate, silent
  → Report — category, description, optional recent in-app messages
  → Optional: leave group / cancel invitation
  → Acknowledgment + limited case status in Safety Center
```

## Copy

- Blocking is not a moderation finding
- Reports reviewed proportionately
- Emergency guidance for imminent danger (not 24/7 monitoring claim)

## Success

Contact ends; user knows next steps for reports.
