# J08 — Two users plan their first gaming session

**Journey:** 19.8 | **Screens:** Conversation → Play invitation → Notifications → Play tab

## User goal

Turn conversation into a realistic first play opportunity.

## Flow

```
From conversation: "Invite to play"
  → Composer: game/platform (required), play now OR up to 3 times, length, voice/text plan, note (optional)
  → Recipient: accept | counter | decline | discuss in chat
  → Reminders (24h, 30m) — user configurable
  → Optional .ics export
  → Session status on Play tab
```

## UX principle

Compact invitation — not appointment software. Paths: **Play now**, **Pick a time**, **Suggest in chat**.

## Safety

- No precise location
- External usernames user-controlled
- Block/report remains available

## Success

Both know what, when, how long, and how they plan to communicate.
