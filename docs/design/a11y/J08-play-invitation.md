# A11y — J08 Play Invitation

## Composer

- Game/platform: required fields marked `aria-required="true"`
- Time options: each slot has `datetime` + visible local TZ label
- "Play now" vs scheduled: radio group with clear labels

## Invitation card (recipient)

- Status not color-only: text label (Proposed, Accepted, etc.)
- Accept / Counter / Decline: distinct button names

## Reminders

- User can disable non-safety reminders in settings; safety-related notices still delivered

## Calendar export

- Link text: "Download calendar file (.ics)" — describes format

## Time zone

- All times show zone abbreviation; SR reads full localized string
