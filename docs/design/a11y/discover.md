# A11y — Discover

## Recommendation cards

- Each card: `article` with heading = display name
- "Why shown" list: visible `ul` with ≥3 items; not `title` tooltip only
- Dismiss/hide: button with accessible name "Hide this gamer from recommendations"

## Density empty state

- Heading explains cohort status plainly
- Demand opt-in: checkbox + submit; success announced via live region

## Filters

- Mobile: filter opens in dialog with focus trap
- Hard vs preferred filters: helper text, not color-only
- Apply/Reset: focus returns to trigger on close

## Pause discovery

- Switch with label "Pause new discovery"
- Confirm if pausing with active intent

## Keyboard

- Card actions reachable without tabbing through entire card text
